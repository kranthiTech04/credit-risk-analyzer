💳 Credit Risk Analyzer

A smart web application that analyzes a user's financial data and predicts whether they are High Risk or Low Risk based on Debt-to-Income ratio and payment history.

---

🚀 Features

- 📊 Calculates Debt-to-Income (DTI) ratio
- ⚠️ Identifies High Risk or Low Risk users
- 🧠 Provides clear explanation for the result
- 🕘 Stores assessment history using LocalStorage
- 🎨 Clean and interactive user interface

---

🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---

📊 How It Works

- If DTI > 40% OR late payments > 2 → 🔴 High Risk
- Otherwise → 🟢 Low Risk
